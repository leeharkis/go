# Lee Harkis - Homepage

A personal homepage with links to my projects and games.

## Projects

- **Play Wordle** - Interactive Wordle game
- **Wordle Solver** - Solver tool for Wordle
- **Play Boggle** - Boggle word game

## Features

- Modern glassmorphism design
- Responsive layout for all devices
- Full-screen background image

## Getting Started

Simply open `index.html` in a web browser, or deploy to GitHub Pages.

## Deployment

To deploy this as a GitHub Pages site:

1. Push this repository to GitHub
2. Go to repository Settings → Pages
3. Select the branch (main) and save
4. Your site will be available at `https://yourusername.github.io/`
